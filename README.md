# SAP User Access Analyzer

## About the Project
It is an Object-Oriented ABAP class (`zcl_user_analyzer`) designed to simulate the retrieval and analysis of user access data in an SAP system. The program identifies and reports active security alerts, specifically focusing on users whose accounts are locked but still retain active authorizations/roles.

## Technical Skills Demonstrated:
* Object-Oriented ABAP (OO-ABAP)
* ABAP Cloud syntax and best practices
* Implementation of `if_oo_adt_classrun` for console output
* Internal table data processing and filtering
